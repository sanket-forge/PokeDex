PokeDex App - Explore Pokémons

Overview

Welcome to the Pokédex App! This application allows users to explore various Pokémon, 
displaying names from the PokéAPI. 
Whether you're a Pokémon enthusiast or just curious about the world of Pokémon, 
this app provides an engaging and informative experience

Features

🗃️ Fetching Data from PokéAPI

- Retrieves Pokémon names directly from the PokéAPI.

🖌️ SwiftUI Compose for UI

- Utilizes swiftUI for a modern, responsive UI.
- Enjoy a seamless and visually appealing user experience.
